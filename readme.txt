== Description ==

自动添加文章目录插件可以自动为你的文章生成一个目录。

插件介绍页面：<a href="http://www.isyeo.com/wordpress-jquery-content-directory/">http://www.isyeo.com/wordpress-jquery-content-directory/</a>

== Installation ==

1. 上传 `jquery_cd.php`, `jquery_cd.css`, and `jquery_cd.js` 指 `/wp-content/plugins/jquery-content-directory/` 目录。 或者通过Wordpress后台进行安装）
2. 激活插件。
3. 配置插件属性以及修改jquery_cd.css中的样式。

== Frequently Asked Questions ==

= Does this plugin work automatically for every theme? =

不是每个主题都能适用该插件，需要你有一点编程的基础，知道如何设置。
